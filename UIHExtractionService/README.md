## Version 1.0 - 12/08/2025
Env: Visual Studio 2022 (64 bits) Version 17.14.10. .Net 8.0

Initial release with core functionalities:
- Windows installer
- The extraction of enumeration starts at system startup and works in the background.
- The status is shown in a tray icon.
- The program can be paused and stopped from the menu by right clicking on the icon.
- Multiple USB Hubs connected simultaneously are supported, but for now, just one in the top level.
